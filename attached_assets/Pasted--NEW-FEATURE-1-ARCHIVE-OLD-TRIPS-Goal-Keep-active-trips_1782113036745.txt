## NEW FEATURE 1: ARCHIVE OLD TRIPS

Goal:
Keep active trips clean while preserving historical trip data.

Requirements:

1. Automatically move trips to "Archived Trips" when:

   * trip end date is more than 6 months in the past.

2. Active trips should display separately from archived trips.

3. Add:

   * Active Trips
   * Archived Trips

4. Archived trips remain fully viewable:

   * itinerary
   * expenses
   * documents
   * places
   * participants

5. Archived trips should be read-only by default.

## FEATURE 2: EXPORT EXPENSES

Goal:
Allow users to export trip expenses.

Requirements:

1. Add Export button inside Expense section.

2. Export formats:

   * CSV

3. Export should include:

   * date
   * expense title
   * category
   * amount
   * payer
   * participants involved in split if any
   * split values
   * notes

---

## FEATURE 3: SHARED TRIPS AND MEMBER MANAGEMENT

Goal:
Allow multiple people to collaborate on the same trip.

Requirements:

1. Add Members section in trip settings.

2. Every trip should have:

   * unique trip code
   * shareable invite code

Example:
MOVE-72Q8

3. Users can:

   * join a trip using a code
   * invite people using a code

4. Trip members should instantly see updates from all participants.

5. Member list should display:

   * profile icon
   * name

---

## FEATURE 4: EXPENSE SPLITTING

Goal:
Allow group expense management.

Requirements:

1. Keep expense splitting inside the existing Expense section.

2. Do NOT create a separate section.

3. Split Expense option should appear ONLY when:

   * trip contains more than one member.

4. If trip has only one member:

   * hide split controls completely.

Split modes:

A. Equal Split

* divide amount equally among selected members

B. Unequal Split

* manually assign amounts per participant

Users must be able to select:

* all members
* specific members only

---
